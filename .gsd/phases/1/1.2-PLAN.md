# Plan 1.2: Applying Theme to Main Containers

## Objective
Podpięcie utworzonego `ThemeManager` pod główne okno aplikacji (`app.py`), w tym Navbar i podstawowe ramki zawartości (`CTKFrame`), zastępując domyślne tło CustomTkinter naszym systemem "Tokyo Night".

## Context
- `ROADMAP.md` (Phase 1)
- `src/kontakt/ui/theme.py`
- `src/kontakt/ui/app.py`
- `src/kontakt/ui/views/navbar.py`
