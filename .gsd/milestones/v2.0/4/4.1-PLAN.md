# Plan 4.1: Settings UI and Model for Seller Data

## Objective
Dodanie modelu bazy danych i interfejsu "Ustawienia", umożliwiających użytkownikowi zapisanie stałych danych podmiotu (Sprzedawcy). Dane te są absolutnie wymagane przez strukturę FA(2) KSeF (`Podmiot1`).
